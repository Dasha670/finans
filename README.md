## Разработка веб-приложения для учёта личных финансов

### О проекте

Данный репозиторий содержит программный код веб-приложения для управления личными финансами. Приложение предназначено для автоматизации учёта финансовых операций физических лиц и предоставления актуальной информации о банковских продуктах.

### Функциональное назначение

Программное средство обеспечивает выполнение следующих функций:

- учёт банковских вкладов с возможностью расчёта процентного дохода (с капитализацией и без);
- контроль задолженностей различных типов (кредиты, займы) и внесение платежей по ним;
- автоматическое обновление курсов валют посредством интеграции с API Центрального банка РФ;
- сбор и актуализация информации о ставках по вкладам ведущих российских банков;
- ведение ленты финансовых советов (блога) с разграничением прав доступа;
- администрирование пользователей и управление правами на публикацию.

### Структура программного средства

Веб-приложение реализовано по классической трёхзвенной архитектуре «клиент-сервер» и включает следующие компоненты:

#### Серверная часть (бэкенд)
- Язык программирования: PHP 7.4+
- СУБД: MySQL 5.7+
- Взаимодействие с базой данных: PDO (PHP Data Objects) с использованием подготовленных выражений
- Интеграция с внешними источниками: API ЦБ РФ, парсинг HTML-страниц финансовых агрегаторов

#### Клиентская часть (фронтенд)
- Языки разметки и стилизации: HTML5, CSS3
- Язык программирования: JavaScript (нативный)
- Асинхронное взаимодействие: Fetch API (AJAX)
